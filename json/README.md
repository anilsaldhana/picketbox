PicketBox JSON
========================
JSON Security (JOSE) from http://datatracker.ietf.org/wg/jose/

Building
-------------------
If you already have Maven 3 installed

	mvn install
	
Documentation
-------------------
For more information, see https://docs.jboss.org/author/display/SECURITY/JSON+Security.
